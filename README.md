Bitezy :
A responsive food delivery web application built using React. Users can browse food items, search dishes, add items to cart, and view their order summary.

You can check live demo:
https://bitezy-9bpam7to7-shreyasingh74s-projects.vercel.app

Features:
- Browse food items by category
- Search food items
- Add and remove items from cart
- Cart total calculation
- Responsive design for mobile and desktop
- Modern React UI

Tech Stack:
- React
- JavaScript
- CSS
- Context API
- React Router
- Vite
- Vercel
